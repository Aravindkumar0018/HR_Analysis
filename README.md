# HR_Analysis
HR Analytics Dashboard project built using Power BI and Tableau to analyze employee attrition, job satisfaction, salary trends, promotion history, and workforce demographics. Provides interactive insights for HR teams through KPIs, filters, drill-downs, and visual storytelling to support data-driven decision-making.

# Tools Used: Power BI | Tableau | Excel | SQL

**The HR Analytics Dashboard** provides a comprehensive analysis of employee data to help HR teams improve workforce planning, retention strategies, and employee satisfaction. This project converts raw HR data into meaningful insights by visualizing key metrics such as **attrition, job satisfaction, promotion history, working years, salary trends, and workforce distribution.**
